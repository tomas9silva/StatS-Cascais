STATS VISION — INSTALAÇÃO COMO APLICAÇÃO

1. Publicar
   - Descompacta este ficheiro ZIP.
   - Coloca index.html, manifest.webmanifest, service-worker.js e a pasta icons na raiz do teu site ou repositório GitHub Pages.
   - A aplicação tem de ser aberta através de HTTPS. Abrir o index.html diretamente a partir da pasta não ativa o modo offline.

2. Instalar no iPhone ou iPad
   - Abre o endereço da StatS Vision no Safari.
   - Carrega em Partilhar.
   - Escolhe "Adicionar ao ecrã principal".
   - Ativa "Abrir como aplicação", se essa opção aparecer, e confirma.

3. Instalar no Mac
   - Abre o endereço no Safari.
   - Escolhe Ficheiro > Adicionar à Dock.

4. Primeira utilização
   - Abre a aplicação uma vez com internet para guardar os ficheiros essenciais.
   - Depois dessa primeira abertura, a aplicação consegue arrancar sem internet.

5. Dados dos jogos
   - Os dados ficam guardados no dispositivo onde a aplicação está instalada.
   - Usa regularmente a exportação de backup da própria aplicação.
   - Instalações em dispositivos diferentes não sincronizam automaticamente entre si.

Conteúdo do pacote
   index.html                 Aplicação StatS Vision
   manifest.webmanifest       Nome, cores, orientação e ícones
   service-worker.js          Arranque e utilização offline
   icons/                     Ícones da aplicação
